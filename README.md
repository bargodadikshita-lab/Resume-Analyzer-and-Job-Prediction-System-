# Resume Analyzer and Job Prediction

An NLP and Machine Learning based web application that analyzes resumes and predicts suitable job roles based on resume content and skills.

## Features
- Resume upload system
- Resume text extraction
- Job prediction using Machine Learning
- Simple and interactive UI

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS/JavaScript
